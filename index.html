The World Wide Web is a collection of Hyper Text Markup Language (HTML) documents.

Major pieces = people (their computer & browser) + the Internet + servers.

HTML is made up of:
- Text content: what you see
- Markup: what it looks like
- References to other documents: e.g. images & videos
- Links to other pages

Definitions

Tag
An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called "void" tags) do not require a closing tag (like the break tag).

Element
An HTML element refers to everything within a set of opening and closing tags.

Attribute
This is a property of an HTML element
